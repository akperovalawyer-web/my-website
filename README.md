# Сторге Мебель

Лендинг мебельного бренда «Сторге Мебель» — мастерская мебели и лестниц из массива по индивидуальным проектам.

## Стек

Статичный сайт без сборки: HTML + CSS + ванильный JS.

## Структура

```
storge-mebel/
├── index.html   — разметка одной страницы со всеми секциями
├── styles.css   — стили (включая адаптив до 480px)
└── script.js    — бургер-меню, маска телефона, валидация формы, reveal-анимации
```

## Секции

1. Шапка с навигацией и CTA
2. Hero-блок с ключевыми метриками
3. О мастерской
4. Форма заявки на проект
5. 8 принципов работы
6. Портфолио проектов
7. Каталог коллекций (10 категорий)
8. Партнёрская программа для дизайнеров
9. Футер с контактами

## Локальный запуск

Откройте `index.html` в браузере — сборка не нужна. Для удобства можно поднять локальный сервер:

```bash
python3 -m http.server 8080
```

## Шрифты

Cormorant Garamond (заголовки) и Inter (текст) подгружаются с Google Fonts.
